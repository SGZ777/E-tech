# Projeto-E-tech
E-tech é um projeto de e-commerce voltado para a área de tecnologia, desenvolvido com o objetivo de simular uma plataforma de venda de produtos tecnológicos. A aplicação apresenta uma vitrine de produtos organizada, permitindo ao usuário visualizar itens, navegar por categorias e ter uma experiência de compra intuitiva.
O projeto foi desenvolvido com foco em design responsivo, usabilidade e organização do layout, buscando reproduzir a estrutura de um e-commerce real. A plataforma faz parte do portfólio e foi criada para a prática de conceitos fundamentais do desenvolvimento web e construção de interfaces comerciais.
As tecnologias utilizadas no desenvolvimento foram HTML, CSS e PHP, além do uso do Git para versionamento de código.
